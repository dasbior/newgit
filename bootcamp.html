Cara membuat list

Membuat daftar berurutan menggunakan Tag "ol = Order list"

Contoh

  1. minuman
  2. makanan
  3. lauk

bisa ditambahkan atribut "type" untuk merubah menjadi angka [ 1,huruf A/a, I/i]

Atribut type huruf besar "A"

  1. jeruk
  2. mangga

Atribut type "a"

  1. jagung
  2. padi

Atribut type huruf romawi "I/i"

  1. padi
  2. gandum

Menggunakan Tag "ul = unorder list"

Contoh

  • home
  • about

Tag ul bisa di tambahkan atribut Type "disc,square,circle"

Contoh atribut "type disc"

  • nasi
  • jagung

Contoh atribut "type Square"

  • budi
  • Tono

Contoh atribut "type Circle"

  • atas
  • bawah

Membuat Definition List dengan Tag "dl"

Berfungsi untuk membuat terminologi dan deskripsi "dl -> dt->dd"
